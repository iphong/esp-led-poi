#  Data Structure References

## Headers


### Pairing set master message
```
#>PAIR*
```

### Pairing set channel message
```
#>PAIR@
```

### Set RGB color for single channel
```
#>CRGB@
```

### Set RGB color for multiple channels
```
#>CRGB+
```

### Upload asset begin
```
- #>FILE^[PATH ...]

```

### Upload asset segment
```
#>FILE+[ADDR 2][DATA ...]
```

### Upload asset end
```
#>FILE$
```
